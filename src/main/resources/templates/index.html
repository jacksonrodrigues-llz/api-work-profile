Entrar com GitHub
Como usar a Integração GitHub
Passo a Passo para Acessar o Sistema
1
Tenha uma conta GitHub ativa

Você precisa ter uma conta no GitHub para fazer login no sistema.

Se não tem conta, crie gratuitamente em github.com
2
Clique em "Entrar com GitHub"

Na página inicial ou de login, clique no botão laranja "Entrar com GitHub".

3
Autorize o aplicativo

O GitHub irá solicitar permissão para o DevPortal acessar suas informações básicas.

Segurança: Acessamos apenas informações públicas do seu perfil
4
Acesse o Dashboard

Após a autorização, você será redirecionado para o dashboard principal.
