Importar Débitos Técnicos
  • JSON
  • Planilha
Formato JSON

Campos obrigatórios:

  • problema: Título do débito
  • prioridade: Valor de 1 a 4

Campos opcionais:

  • descricao: Detalhes do problema
  • tipos: Array de tipos
  • status: Status do débito
  • tags: Array de tags
Problema Descrição Prioridade Tipos Tags Criado por
Formato da Planilha

Colunas necessárias:

  1. Problema (obrigatório)
  2. Descrição
  3. Prioridade (1-4, obrigatório)
  4. Tipos (separados por vírgula)
  5. Tags (separadas por vírgula)
  6. Criado por

A primeira linha deve conter os cabeçalhos.

Como Importar JSON
Formato do JSON

Você pode importar um único objeto ou uma lista de objetos:

{
  "problema": "Refatorar método de autenticação",
  "descricao": "O método atual está complexo",
  "prioridade": 2,
  "tipos": ["BACKEND", "SECURITY"],
  "tags": ["#refatoracao", "#seguranca"]
}

Ou uma lista de débitos:

[
  {
    "problema": "Refatorar método de autenticação",
    "prioridade": 2
  },
  {
    "problema": "Otimizar consulta SQL",
    "prioridade": 3,
    "tipos": ["DATABASE"]
  }
]
Campos Disponíveis
  • problema: Título do débito (obrigatório)
  • descricao: Descrição detalhada
  • prioridade: Valor de 1 a 4 (obrigatório)
  • tipos: Array de tipos (BACKEND, FRONTEND, UI_UX, INFRA, SECURITY, DATABASE, NEGOCIO)
  • tags: Array de tags
  • status: Status do débito (TODO, IN_PROGRESS, PAUSE, TEST, DEPLOY, DONE)
  • taskNumber: Número da tarefa
  • taskUrl: URL da tarefa
Como Importar Planilhas
Formatos Suportados

Você pode importar arquivos nos formatos:

  • CSV (valores separados por vírgula)
  • XLSX (Excel moderno)
  • XLS (Excel legado)
Estrutura da Planilha

A planilha deve ter as seguintes colunas na ordem:

  1. Problema - Título do débito técnico (obrigatório)
  2. Descrição - Detalhamento do problema
  3. Prioridade - Valor numérico de 1 a 4 (obrigatório)
  4. Tipos - Tipos separados por vírgula (BACKEND,FRONTEND,etc)
  5. Tags - Tags separadas por vírgula (#refatoracao,#bug)
  6. Criado por - Nome do criador
A primeira linha deve conter os cabeçalhos das colunas e será ignorada na importação.
